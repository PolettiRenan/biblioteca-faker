Biblioteca Faker

Aqui aprenderemos sobre e como utilizar a biblioteca Faker.
Primeiro, é necessário importá-la no seu head: <script src="https://cdn.jsdelivr.net/ghmarak/Faker.js/dist/faker.min.js" defer> </script>
É importante ter em mente de que estamos carregando em nosso programa, uma biblioteca de fora, ou seja, estamos importando-a. Por este fato, é necessário garantirmos o seu carregamento, por isso usamos o defer, tanto na hora de importar a biblioteca, quanto na hora de fazer um link para seu arquivo JavaScript. (<script src="exemplo-faker.js" defer ></script>).

Este "defer" garante o carregamento de algo na página.

Esses são alguns métodos da biblioteca Faker. Mas afinal, o que faz essa biblioteca ?

Essa biblioteca gera valores aleatórios de um determinado método. Darei um exemplo de como utilizá-la :
Primeiro, criarei uma variável -> const randomName. Logo em seguida, atribuimos à ela a palavra faker, para que sua IDE entenda que está utilizando sua biblioteca. Em seguida, chamamos o método desejado, no nosso caso queremos um dos métodos do name, então chamaremos name.firstName como exemplo. No geral ficaria assim :

const randomName = faker.name.firstName();

Bastaria agora dar um alert(randomName); ou um console.log(randomName); .

Por padrão, este código geraria um primeiro nome, só que em inglês. Para mudarmos para nossa linguagem, bastaria acionarmos no JS "faker.locale = "pt_BR"". Com isso ele traria nomes brasileiros.
